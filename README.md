# tril

Streaming github trend repos on Slack.

<img src="https://github.com/gotokatsuya/tril/blob/master/demo.png" alt="demo" width=600 height=400>

## Setup

```shell
export TRIL_SLACK_WEBHOOK_URL=""
export TRIL_DB_PATH="/PATH/tril.db"
```

```shell
$ yarn install
```

## Launch

```shell
$ yarn start
```
